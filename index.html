BlueWeather

BlueWeather is a Personal Weather Station Server that can collect weather data and deliver it directly to your device.

It is a highly extensible application to allow for the possibility of any hardware to work with BlueWeather and to allow for custom data processing.

Gemcaps

Gemcaps is an optimized server for the gemini protocol. I hope to make this software easy to configure, but alsohighly configurable.

The gemini protocol aims to be simple and unextensible. I like that, however, I want to have servers be extensible. It would be nice to have a central server that acts as a proxy to other servers in the same domain. To allow the central server to cache pages to allow for quicker response times. Along with so much more.
